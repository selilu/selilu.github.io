Coding Projects

Overview

The goal of this website is to create a responsive and easy-to-navigate platform that showcases policies, opportunities, and classes available to first-year engineering students. This website serves as an information hub with contacts, information, and highlights of the College of Engineering. Built with a target audience of other educational professionals, students, and parents alike, the website is formal and professional. Highlighted is the E100 section which was visually redesigned to attract potential students with brighter colors, shapes, and modern elements.

Key Features

  • Responsive Design
    Optimized for desktop and mobile based on analytics that revealed these were the most commonly used screen sizes for users.
  • Technologies Used
    Built with Divi Web Builder on Wordpress, CSS, HTML, Google Analytics, Google Tag
  • Class Display and Visuals
    Filterable classes with hover descriptions for easy viewability. Visual features such as scrolling external partners list, buttons, hover effects to engage users
