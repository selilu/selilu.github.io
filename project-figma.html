NFT Trading Website
Figma Design Mock-up

Overview

The goal of this website design mock-up is to learn how to use Figma and get more familiar with the tools and their usage. While building the design, UI/UX principles were learned and applied such as text hierarchy, card layouts, visual consistency, and clear call-to-action elements.

Key Features

  • Technologies Used
    Figma
  • Testimonials
    Profile photos with blend applied to match the color scheme of the website and make it more cohesive.
  • Modern Visuals
    Rounded buttons and boxes soften the look of the website while matching the style of the images.
