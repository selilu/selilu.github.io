GOGO Tennis
AR Tennis Game

Overview

The goal of this project was to redesign an existing AR tennis game with brighter colors, shapes, and modern elements that appear more engaging and cohesive to users.

Role: UX Designer and Researcher in charge of completing analysis to discover pain points and weaknesses of previous design, assist in determining new game theme, create reusable UI components, and more.

Key Features

  • Responsive Design
    Designed for mobile and desktop use.
  • Technologies Used
    Components and UI created with Figma, 3D modeling using Blender
